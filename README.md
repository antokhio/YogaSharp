Facebook Yoga bindings for C#

Available via NuGet:
```
nuget install antokhio.YogaSharp
```

This was assembled quite a while ago, based on original [repo](https://github.com/Haselnussbomber/YogaSharp) by [Haselnussbomber](https://github.com/Haselnussbomber). Slightly modified to publish NuGet.
Time passed, and this repository needs maintenance work, such as:

* Automated release pipeline
* Fetch updates from facebook/yoga repo

If you find this repository useful, think about contributing. 

If you want to submit a breaking change please first fire an [issue](https://github.com/antokhio/YogaSharp/issues).


Contributions are welcome.
